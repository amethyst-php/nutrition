Railken\Amethyst\Fakers\NutritionFaker

The faker can be used for testing or seeding.

Create a new entity using the faker

use Railken\Amethyst\Fakers\NutritionFaker;

$result = $manager->create(NutritionFaker::make()->parameters());

Extend the class

Create the new faker in app/Fakers/NutritionFaker

namespace App\Fakers;

use Railken\Amethyst\Fakers\NutritionFaker as Faker;

class NutritionFaker extends Faker {
    // ...
}

Update the file configs/amethyst.nutrition with the new class

return [
    'data' => [
        'nutrition' => [
            'faker' => App\Fakers\NutritionFaker::class,
        ],
    ]
];
